## 2. –§–æ—Ä–º–∞–ª—å–Ω–æ–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ OBSFitness

–û–±–æ–∑–Ω–∞—á–∏–º:

- T\_total ‚Äî –¥–ª–∏–Ω–∞ –Ω–∞–±–ª—é–¥–µ–Ω–∏—è (–∫–æ–ª-–≤–æ —à–∞–≥–æ–≤ —Å–∏–º—É–ª—è—Ü–∏–∏);
- `knowledge` ‚Äî —Ñ–∏–Ω–∞–ª—å–Ω–æ–µ SemanticState –Ω–∞–±–ª—é–¥–∞—Ç–µ–ª—è –ø–æ—Å–ª–µ T\_total —à–∞–≥–æ–≤.

### 2.1. –ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã —Ñ–∏—Ç–Ω–µ—Å–∞

–û–ø—Ä–µ–¥–µ–ª–∏–º —à–µ—Å—Ç—å –∫–æ–º–ø–æ–Ω–µ–Ω—Ç:

1. \(F_\text{field}\) ‚Äî –∫–∞—á–µ—Å—Ç–≤–æ –∏–∑–≤–ª–µ—á–µ–Ω–∏—è –ø–æ–ª–µ–≤–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è.

3. \(F_M\) ‚Äî —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–µ ¬´–º–∞—Å—Å—ã¬ª / —ç–Ω–µ—Ä–≥–∏–∏ M (–∞–≥—Ä–µ–≥–∞—Ç –ø–æ Œ©‚Äë–æ–±—ä–µ–∫—Ç–∞–º).

5. \(F_\text{grav}\) ‚Äî –∏–∑–≤–ª–µ—á–µ–Ω–∏–µ –≥—Ä–∞–≤–∏—Ç–∞—Ü–∏–æ–Ω–Ω–æ–≥–æ –∑–∞–∫–æ–Ω–∞ (–∫–æ—Ä—Ä–µ–ª—è—Ü–∏—è a vs ‚àí‚àáŒ¶).
6. \(F_\text{prob}\) ‚Äî —Å–æ–≥–ª–∞—Å–æ–≤–∞–Ω–Ω–æ—Å—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π (–µ—Å–ª–∏ –ø–æ–∫–∞ –Ω–µ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–æ, –º–æ–∂–Ω–æ –∑–∞–¥–∞—Ç—å –ø—Ä–æ—Ç–æ—Ç–∏–ø).

–ö–∞–∂–¥—ã–π \(F_\cdot \in [0,1]\).

### 2.2. \(F_\text{field}\): –ø–æ–ª–µ–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ

–í `observer_demo` –≤—ã –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –õ–∞–≥—Ä–∞–Ω–∂–∏–∞–Ω–∞:

\[
\partial_t^2 \phi = \kappa \nabla^2 \phi - m^2 \phi - \lambda \phi^3 + \dots
\]

–∏ —Å—á–∏—Ç–∞–µ—Ç–µ R¬≤ (–∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç –¥–µ—Ç–µ—Ä–º–∏–Ω–∞—Ü–∏–∏) –ø–æ –ª–∏–Ω–µ–π–Ω–æ–π/–Ω–µ–ª–∏–Ω–µ–π–Ω–æ–π —Ä–µ–≥—Ä–µ—Å—Å–∏–∏.

–ü—É—Å—Ç—å:

- R¬≤\_t ‚Äî –∑–Ω–∞—á–µ–Ω–∏–µ R¬≤ –Ω–∞ t‚Äë–º –æ–±–Ω–æ–≤–ª–µ–Ω–∏–∏ —Å–µ–º–∞–Ω—Ç–∏–∫–∏,
- \(R^2_\text{mean}\) ‚Äî —Å—Ä–µ–¥–Ω–µ–µ –ø–æ –ø–æ—Å–ª–µ–¥–Ω–∏–º K —à–∞–≥–∞–º (–∏–ª–∏ –ø–æ –≤—Å–µ–π –∏—Å—Ç–æ—Ä–∏–∏, –µ—Å–ª–∏ –µ—ë –º–∞–ª–æ).

–¢–æ–≥–¥–∞:

\[
F_{\text{field}} = \max(0, \min(1, R^2_\text{mean})).
\]

–ï—Å–ª–∏ –≤—ã —É—Å–µ—á—ë—Ç–µ R¬≤ –Ω–∞ [0,1], —ç—Ç–æ —Å—Ä–∞–∑—É –¥–∞—ë—Ç –∫–æ—Ä—Ä–µ–∫—Ç–Ω—É—é –Ω–æ—Ä–º–∏—Ä–æ–≤–∫—É.

–ù–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ –≤ –∫–æ–¥–µ:

```python
R2_vals = [entry.R2 for entry in knowledge.field_history]
if not R2_vals:
    F_field = 0.0
else:
    R2_mean = np.mean(R2_vals[-K:])  # K=5 –∏–ª–∏ –≤—Å–µ, –µ—Å–ª–∏ len<K
    F_field = max(0.0, min(1.0, R2_mean))
```

---

### 2.3. \(F_Q\): —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–µ –∑–∞—Ä—è–¥–∞

–ü—É—Å—Ç—å:

- Q(t) ‚Äî —Å—É–º–º–∞—Ä–Ω—ã–π –Ω–∞–±–ª—é–¥–∞–µ–º—ã–π –∑–∞—Ä—è–¥ –≤ IFACE –≤ –º–æ–º–µ–Ω—Ç t,
- Q(0) ‚Äî –Ω–∞—á–∞–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

–û–ø—Ä–µ–¥–µ–ª–∏–º –Ω–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –Ω–∞—Ä—É—à–µ–Ω–∏–µ:

\[
\delta_Q(t) = \frac{|Q(t) - Q(0)|}{1 + |Q(0)|}
\]

–∏ –≤–æ–∑—å–º—ë–º –º–∞–∫—Å–∏–º—É–º –ø–æ –≤—Ä–µ–º–µ–Ω–∏:

\[
\delta_Q^\text{max} = \max_{0\le t\le T_\text{total}} \delta_Q(t).
\]

–§–∏—Ç–Ω–µ—Å:

\[
F_Q = \exp\left(-\frac{\delta_Q^\text{max}}{\sigma_Q}\right),
\]

–≥–¥–µ \(\sigma_Q > 0\) ‚Äî –ø–∞—Ä–∞–º–µ—Ç—Ä –∏–∑ `OBSFitnessConfig` (–Ω–∞–ø—Ä–∏–º–µ—Ä, 0.1).

–ö–æ–¥:

```python
delta_Q = max( abs(Q_t - Q_0) / (1.0 + abs(Q_0)) for Q_t in Q_history )
F_Q = np.exp(-delta_Q / cfg.sigma_Q)
F_Q = float(max(0.0, min(1.0, F_Q)))
```

### 2.4. \(F_M\): —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏–µ –º–∞—Å—Å—ã/—ç–Ω–µ—Ä–≥–∏–∏

–ê–Ω–∞–ª–æ–≥–∏—á–Ω–æ –¥–ª—è –º–∞—Å—Å—ã:

- M(t) ‚Äî —Å—É–º–º–∞—Ä–Ω–∞—è ¬´–º–∞—Å—Å–∞¬ª Œ©‚Äë–æ–±—ä–µ–∫—Ç–æ–≤ –∏–ª–∏ –∏–Ω—Ç–µ–≥—Ä–∞–ª H\_micro –ø–æ —Ä–µ—à—ë—Ç–∫–µ.

–û–ø—Ä–µ–¥–µ–ª—è–µ–º:

\[
\delta_M(t) = \frac{|M(t) - M(0)|}{1 + |M(0)|},\quad
\delta_M^\text{max} = \max_t \delta_M(t),
\]

\[
F_M = \exp\left(-\frac{\delta_M^\text{max}}{\sigma_M}\right),
\]

—Å \(\sigma_M\) ‚Äî —Ç–∏–ø–∞ cfg.sigma_M (–µ—Å–ª–∏ —Ö–æ—Ç–∏—Ç–µ, –º–æ–∂–Ω–æ –≤–∑—è—Ç—å —Ç—É –∂–µ œÉ, —á—Ç–æ –∏ –¥–ª—è Q).

### 2.5. \(F_{OT}\): Observation Time

–°–æ–≥–ª–∞—Å–Ω–æ Meaning_v1 –∏ –≤–∞—à–µ–π —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏:

- t\_OT ‚Äî –º–æ–º–µ–Ω—Ç (–≤ —à–∞–≥–∞—Ö e–≤–æ–ª—é—Ü–∏–∏), –∫–æ–≥–¥–∞ –æ—Ü–µ–Ω–∫–∏ –ø–æ–ª–µ–≤–æ–≥–æ —É—Ä–∞–≤–Ω–µ–Ω–∏—è (–∏/–∏–ª–∏ –∑–∞–∫–æ–Ω–æ–≤) –≤–æ—à–ª–∏ –≤ —Å—Ç–∞–±–∏–ª—å–Ω—ã–π –¥–∏–∞–ø–∞–∑–æ–Ω –∏ –æ—Å—Ç–∞—é—Ç—Å—è —Ç–∞–º;
- –µ—Å–ª–∏ —Å—Ç–∞–±–∏–ª–∏–∑–∞—Ü–∏–∏ –Ω–µ –±—ã–ª–æ ‚Äî –º–æ–∂–Ω–æ –ª–∏–±–æ:
  - —Å—Ç–∞–≤–∏—Ç—å F\_{OT}=0,
  - –ª–∏–±–æ —à—Ç—Ä–∞—Ñ–æ–≤–∞—Ç—å –ø–æ ¬´—É–¥–µ—Ä–∂–∞–Ω–∏—é¬ª –æ—à–∏–±–∫–∏.

–ù–æ—Ä–º–∏—Ä–æ–≤–∫–∞:

\[
F_{OT} =
\begin{cases}
0, & \text{–µ—Å–ª–∏ } t_{OT}\ \text{–Ω–µ –æ–ø—Ä–µ–¥–µ–ª—ë–Ω}, \\
\exp\left(-\frac{t_{OT}}{T_\text{scale}}\right), & \text{–∏–Ω–∞—á–µ},
\end{cases}
\]

–≥–¥–µ \(T_\text{scale} = \text{cfg.T\_scale\_fraction} \cdot T_\text{total}\).

–ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ü–∏—è:

- –µ—Å–ª–∏ OBS —Å—Ç–∞–±–∏–ª–∏–∑–∏—Ä–æ–≤–∞–ª –∑–∞–∫–æ–Ω —Ä–∞–Ω–æ (t\_OT ‚â™ T\_total) ‚Üí F\_{OT} –±–ª–∏–∑–∫–æ –∫ 1;
- –µ—Å–ª–∏ –ø–æ–∑–¥–Ω–æ ‚Üí F\_{OT} –±–ª–∏–∂–µ –∫ 0;
- –µ—Å–ª–∏ –≤–æ–æ–±—â–µ –Ω–µ —Å—Ç–∞–±–∏–ª–∏–∑–∏—Ä–æ–≤–∞–ª ‚Üí 0.

–ö–æ–¥:

```python
    F_OT = 0.0
else:
    T_scale = cfg.T_scale_fraction * T_total
    F_OT = np.exp(- knowledge.t_OT / max(1.0, T_scale))
    F_OT = float(max(0.0, min(1.0, F_OT)))
```

### 2.6. \(F_\text{grav}\): –≥—Ä–∞–≤–∏—Ç–∞—Ü–∏—è

–ï—Å–ª–∏ —É –≤–∞—Å —É–∂–µ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç —Å –ø–∞–¥–µ–Ω–∏–µ–º —á–∞—Å—Ç–∏—Ü –∏ –æ—Ü–µ–Ω–∫–æ–π Œ≥, –≤—ã –º–æ–∂–µ—Ç–µ —Ö—Ä–∞–Ω–∏—Ç—å:

- corr\_grav ‚Äî –∫–æ—Ä—Ä–µ–ª—è—Ü–∏—è –º–µ–∂–¥—É –Ω–∞–±–ª—é–¥–∞–µ–º—ã–º —É—Å–∫–æ—Ä–µ–Ω–∏–µ–º a –∏ (–º–∏–Ω—É—Å) –≥—Ä–∞–¥–∏–µ–Ω—Ç–æ–º —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ–≥–æ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª–∞ ‚àáŒ¶.

–ï—Å–ª–∏ corr\_grav –Ω–µ—Ç –∏–ª–∏ —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç–∞ –ø–æ–∫–∞ –Ω–µ—Ç ‚Äî –º–æ–∂–Ω–æ —Å—Ç–∞–≤–∏—Ç—å:

### 2.7. \(F_\text{prob}\): –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏

–ù–∞ –ø–µ—Ä–≤–æ–º —à–∞–≥–µ, –ø–æ–∫–∞ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–Ω–æ–µ –æ–∫–Ω–æ —É –≤–∞—Å –≤ –∑–∞—á–∞—Ç–∫–µ, –º–æ–∂–Ω–æ –∑–∞–¥–∞—Ç—å –ø—Ä–æ—Å—Ç–æ–π –∫—Ä–∏—Ç–µ—Ä–∏–π:

- OBS —Å—Ç—Ä–æ–∏—Ç –¥–ª—è –Ω–µ–∫–æ—Ç–æ—Ä–æ–≥–æ –¥–∏—Å–∫—Ä–µ—Ç–Ω–æ–≥–æ —Å–æ–±—ã—Ç–∏—è (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Ä–∞—Å–ø–∞–¥/–∞–Ω–Ω–∏–≥–∏–ª—è—Ü–∏—è Œ©‚Äë—á–∞—Å—Ç–∏—Ü) –æ—Ü–µ–Ω–∫—É –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ pÃÇ;

  \[

  \quad

  \]

–ï—Å–ª–∏ —ç—Ç–æ –Ω–µ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω–æ ‚Äî F\_prob=0 —Å–µ–π—á–∞—Å, —Å –ø–µ—Ä—Å–ø–µ–∫—Ç–∏–≤–æ–π —Ä–∞–∑–≤–∏—Ç—å –ø–æ–∑–∂–µ.

## 3. –ò—Ç–æ–≥–æ–≤–∞—è —Ñ–æ—Ä–º—É–ª–∞ OBSFitness

–°–æ–±–∏—Ä–∞–µ–º –∫–æ–º–ø–æ–Ω–µ–Ω—Ç—ã –≤ —Å—Ç—Ä—É–∫—Ç—É—Ä—É:

```python
    sigma_field=0.1,   # –º–æ–∂–Ω–æ –Ω–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å, –µ—Å–ª–∏ R¬≤ —É–∂–µ –Ω–æ—Ä–º–∞–ª–∏–∑–æ–≤–∞–Ω
    sigma_Q=0.1,
```

```python
class OBSFitness:

    def __init__(self, config: OBSFitnessConfig):
        self.cfg = config

    def evaluate(self, observer, T_total: int) -> tuple[float, OBSFitnessComponents]:
        kn = observer.knowledge

        # 1) Field
        R2_vals = [e.R2 for e in kn.field_history]
        if R2_vals:
            R2_mean = np.mean(R2_vals[-min(5, len(R2_vals)):])
            F_field = float(max(0.0, min(1.0, R2_mean)))
        else:
            F_field = 0.0

        # 2) Charge
        Q_hist = kn.Q_history  # —Å–ø–∏—Å–æ–∫ Q(t)
        if Q_hist:
            Q0 = Q_hist[0]
            delta_Q = max(abs(Q - Q0) / (1.0 + abs(Q0)) for Q in Q_hist)
            F_Q = float(np.exp(-delta_Q / self.cfg.sigma_Q))
            F_Q = max(0.0, min(1.0, F_Q))
        else:
            F_Q = 0.0

        # 3) Mass
        M_hist = kn.M_history
        if M_hist:
            M0 = M_hist[0]
            delta_M = max(abs(M - M0) / (1.0 + abs(M0)) for M in M_hist)
            F_M = float(np.exp(-delta_M / self.cfg.sigma_M))
            F_M = max(0.0, min(1.0, F_M))
        else:
            F_M = 0.0

        # 4) Observation Time
        if kn.t_OT is None:
            F_OT = 0.0
        else:
            T_scale = max(1.0, self.cfg.T_scale_fraction * T_total)
            F_OT = float(np.exp(-kn.t_OT / T_scale))
            F_OT = max(0.0, min(1.0, F_OT))

        # 5) Gravity
        if kn.gravity_corr is not None:
            F_grav = float(max(0.0, min(1.0, kn.gravity_corr)))
        else:
            F_grav = 0.0

        # 6) Probabilities
        if kn.prob_error is not None:
            eps_p = kn.prob_error  # |p_emp - p_hat|
            F_prob = float(np.exp(-eps_p / self.cfg.sigma_prob))
            F_prob = max(0.0, min(1.0, F_prob))
        else:
            F_prob = 0.0

        comps = OBSFitnessComponents(
            fitness_field=F_field,
            fitness_Q=F_Q,
            fitness_mass=F_M,
            fitness_OT=F_OT,
            fitness_gravity=F_grav,
            fitness_prob=F_prob,
        )

        # –ê–≥—Ä–µ–≥–∞—Ü–∏—è
        w = self.cfg
        num = (
            w.w_field   * F_field +
            w.w_Q       * F_Q +
            w.w_mass    * F_M +
            w.w_OT      * F_OT +
            w.w_gravity * F_grav +
            w.w_prob    * F_prob
        )
        den = (w.w_field + w.w_Q + w.w_mass + w.w_OT + w.w_gravity + w.w_prob)
        F_total = 0.0 if den == 0.0 else float(num / den)

        return F_total, comps
```
